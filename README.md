# 可视计算与交互概论课程作业

## 📖 简介
本项目是《可视计算与交互概论》课程的大作业，主要围绕 **BVH 动作数据加载、骨架动画播放、以及 BVH 到角色模型的动作重定向与蒙皮显示** 展开。

项目基于课程提供的 **VCX 框架** 实现，核心目标是完成从动作捕捉数据到角色动画展示的基本流程，包括 **BVH 文件解析、骨架层级构建、关节姿态更新、动作重定向，以及基于 GPU Skinning 的角色蒙皮渲染**。通过本项目，我们对角色动画系统中的关键环节进行了实践，包括骨架运动表示、不同骨架之间的映射关系、动画驱动角色模型的方法，以及实时渲染中的蒙皮处理流程。

整个项目既关注动画数据本身的读取与播放，也关注如何将原始动作自然地迁移到具体角色模型上，从而实现从“骨架动画”到“角色动画”的完整展示效果。

## 🌟 主要内容
本项目主要包含两个部分，对应课程作业中的核心功能实现：

- **BVH Loader & Player**  
  解析 BVH 文件中的骨架层级结构、关节偏移信息和逐帧运动数据，并据此构建骨架动画系统，实现 BVH 动作的可视化播放。

- **BVH -> glTF Retargeting**  
  将 BVH 骨架中的动作数据映射到 glTF 角色模型的骨架上，使角色能够跟随原始 BVH 动作进行运动，并展示动作重定向的结果。

在此基础上，项目还进一步完成了以下内容：

- **GPU Skinning**  
  使用线性混合蒙皮（Linear Blend Skinning, LBS）方法，在 GPU 端对角色网格进行蒙皮变换，实现角色模型随骨骼动画实时变形。

- **骨架与角色联合显示**  
  支持同时显示 BVH 原始骨架和绑定后的角色模型，便于对比观察动作播放与重定向的效果。

- **动画结果可视化**  
  通过图形界面直观展示骨架运动、模型姿态变化和整体动画效果，方便分析不同动作数据在角色上的呈现结果。

## 🛠️ 实验环境
本项目使用 C++ 进行开发，并基于课程提供的图形框架完成实现。主要环境如下：

- **编程语言**：C++
- **图形框架**：OpenGL（基于 VCX Engine）
- **构建工具**：xmake
- **角色模型文件**：`assets/models/character.glb`
- **动作数据文件**：`assets/motions/*.bvh`

建议在已经正确配置好图形环境和编译工具链的本地机器上运行本项目。
